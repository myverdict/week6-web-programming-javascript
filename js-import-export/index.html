Modules in client-side JavaScript

This page has two JavaScript files, script.js and data.js.
script.js is the main code file, and it would like to import data.js.

  • In the index.html file include only script.js and include type="module" attribute in the script tag of the head section.
    A module is a function or group of similar functions.

  • data.js needs to export whatever other script files will need. Examples MDN Web Docs: export

  • script.js will import whatever it needs from data.js. Examples MDN Web Docs: import

  • To view this page, you'll need to serve it from a web server. WebStorm will do this for you, when you open a page from the WebStorm IDE using the browser icons in the top left.

  • Not using WebStorm? Install Node.js and then run npx http-server from the directory your HTML file is in, to start a basic HTTP server.
    You'll see output from the command with a URL you can use to view your page.
    It will be easier if your file is already called index.html otherwise add the html file name to the end of the URL.
